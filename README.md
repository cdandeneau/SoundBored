SoundBored

A full-stack application for ranking Spotify songs, artists, and albums on tier lists.

Tech Stack

- Frontend: Next.js 14+ (App Router)
- Database: Supabase (PostgreSQL)
- API: Spotify Web API
- Styling: Tailwind CSS
- Authentication: Spotify OAuth + Supabase Auth

Getting Started

Prerequisites

- Node.js 18+ installed
- A Supabase account ([supabase.com](https://supabase.com))
- A Spotify Developer account ([developer.spotify.com](https://developer.spotify.com))
